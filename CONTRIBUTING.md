# Contributing

This repository holds the SwiftUI study sessions.

## Authoring flow

1. Create a new file in `_sessions/` using `templates/session-template.md`.
2. Add any images or attachments under `assets/sessions/<slug>/`.
3. Fill in front matter first: track, summary, goals, takeaways, Apple links, homework.
4. Write the session body in Markdown.
5. Push to `master` and let GitHub Pages deploy the update.

## Rules

- Keep one session per Markdown file.
- Prefer Apple official documentation as the main outbound reference.
- Keep code blocks runnable and trimmed to the teaching point.
