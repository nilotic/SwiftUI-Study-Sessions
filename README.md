# SwiftUI Study

GitHub Pages skeleton for a durable, searchable SwiftUI study archive.
New session updates should be authored here directly.

## Structure

- `_sessions/`: session pages rendered as `/sessions/<slug>/`
- `_layouts/`: site and session templates
- `_data/tracks.yml`: learning tracks used across the site
- `assets/`: styles and small client-side behavior
- `.github/workflows/pages.yml`: GitHub Pages deployment workflow

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

## Next step
